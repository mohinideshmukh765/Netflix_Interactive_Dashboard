# Netflix_Interactive_Dashboard
This project is an interactive Power BI dashboard built using the Netflix dataset.
It focuses on transforming raw data into meaningful visual insights through data modeling, DAX, and dashboard design.

The dashboard helps analyze content distribution, trends, and patterns across Netflix movies and TV shows.
#

## 🛠️ Tech Stack

**Tool: Power BI**

**Language: DAX (Data Analysis Expressions)**

**Data Processing: Power Query**
#

## 📂 Dataset

**File: netflix data.csv**

**Source: Kaggle**

**Link: https://www.kaggle.com/datasets/victorsoeiro/netflix-tv-shows-and-movies?**
#

## ⚙️ Project Workflow

**Data Loading into Power BI**

**Data Cleaning using Power Query**

**Data Modeling**

**DAX Calculations**

**Dashboard Design**

**Insight Generation**
